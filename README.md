# Vetriver — Trade / Brochure Site

Cream + green-ink B2B brochure site. No cart, no backend — every call-to-action is a
phone call or a WhatsApp link. Static, single HTML file.

See [NOTES.md](NOTES.md) for the design rationale and what to confirm before launch
(placeholder reviews, hours, map pin).

## Run it locally

```bash
open index.html   # no build, no server
```
